# UI Patterns

## DO
- Use max-w-7xl or max-w-[1400px]
- Use grid with responsive columns
- Use gap instead of margin hacks
- Use flex ONLY for small components

## DON'T
- Don't stretch containers full width
- Don't use position:absolute for layout
- Don't hardcode widths like 1000px
- Don't leave large empty whitespace

## 4K Behavior
- Increase columns, not spacing
- Keep content centered
